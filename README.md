# Datasets

A public collection of curated datasets across multiple domains for data analysis, experimentation, and reproducible research.

The datasets are organized by industry and data modality, and are designed for easy access from tools such as Google Colab without requiring authentication or external storage mounts.

## Structure

datasets/
├── finance/
├── healthcare/
├── ecommerce/
├── text/
├── images/
└── time_series/

## Notes
- All datasets are public and non-sensitive
- Files may be provided in CSV and/or Parquet formats
- Individual datasets may include additional documentation and metadata
